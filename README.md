# 24 часа к себе

Интерактивный сайт с ежедневными вызовами для саморазвития: баллы, серия дней, календарь прогресса и специальные предложения.

**Сайт:** [galileya2008-byte.github.io/poslanie](https://galileya2008-byte.github.io/poslanie/)

## Как открыть локально

Откройте `index.html` в браузере или запустите любой простой HTTP-сервер в папке проекта.

## Админ-панель

Нажмите 5 раз на бейдж «Интерактив для саморазвития» или откройте сайт с параметром `?admin`.
